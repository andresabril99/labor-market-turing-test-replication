# Labor Market Turing Test - Replication Package

This repository contains the replication package for the paper *Developing and assessing a scale for measuring human–AI
alignment in hiring decisions*.

The code builds the processed datasets from the cleaned raw inputs and reproduces the figures and tables used in the paper.
